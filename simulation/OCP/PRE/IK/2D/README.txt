The average marker error is acceptable, but the maximum marker
error is a bit high. Where is it coming from?

Though the markers perfectly fit the 3d model, this is not the 
case in 3D: when the subject slightly rotates their arms/wrists 
the plug-in-gait markers move in a way that is *not* consistent 
with 2D rigid body mechanics. Hence the large errors on the 
shoulders (they swoop forward) and the upper arm.
